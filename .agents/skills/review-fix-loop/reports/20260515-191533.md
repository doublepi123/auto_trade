## 最终建议

1. **PBKDF2 salt**: 在下一个版本中生成随机 salt 存储到 `data/credential_salt.bin`，并提供迁移脚本
2. **密钥路径**: 改用环境变量 `AUTO_TRADE_DATA_DIR` 或绝对路径
3. **Kill Switch**: 考虑增加重启确认或二次确认机制
4. **订单状态轮询**: 实现 broker webhook 或定时轮询，使用实际成交价计算 PnL
5. **做空 side 值**: 确认 Longbridge SDK 对做空订单的 side 枚举值
6. **数据迁移**: 编写一次性脚本清理 DB 中加密的空字符串
